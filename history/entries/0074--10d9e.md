## Approach

Output-neutral WORK reduction in the Context Tree Weighting model. The depth-48 CTW walk evaluates exp/ln on every node every bit, but on the deep tail almost every node is empty or first-touch, where those transcendentals have closed-form results. Three bit-exact shortcuts: (1) in predict, alpha = 1/(1+e^arg) is exactly 0.5 when arg==0 (empty node with empty children) since e^0==1; (2) in update, the KT log p_obs.ln() is the precomputed ln(0.5) whenever n0==n1 (p_obs is then exactly 0.5); (3) ln_add(a,b) with a==b is a+ln(2) (its exp(0) term is exactly 1), the common case for first-touch chains where both sides are 2*ln(0.5). Each branch returns a bit-identical f64, so output is byte-for-byte unchanged on all six corpus files (SCORE 572423) and all 9 round-trip tests pass.
