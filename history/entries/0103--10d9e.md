## Approach

Adds a 33rd layer-1 specialist selected by cross-match agreement: at each bit it counts how many of the six match models (order-2..14 anchors) are active, still consistent with the partial byte, and agree on the SAME next bit, crossing that count (0..7) with the agreed bit and the bit position (512 rows). Byte-level consensus between independent match anchors is a much stronger repeat-regime signal than any single match length — six agreeing anchors nearly guarantee the bit, while a lone active match is far weaker; the existing selectors only see individual lengths/flags. Biggest gains on the match-driven files (nci 13109->13100, binary 227861->227842, struct 7302->7299). Local SCORE 569776 -> 569744 (-32). 9/9 round-trip lossless; only src/algorithm/ touched.
